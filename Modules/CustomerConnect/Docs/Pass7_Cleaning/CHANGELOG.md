# Titan Connect – Pass 7 (Cleaning Verticalisation)

## What this pass does
Tailors Titan Connect (CustomerConnect) for **cleaning businesses** (Clean Smart) by updating copy, defaults, and campaign presets.

## Key changes
- Cleaning-friendly campaign templates and labels
- Adds two new preset campaign types:
  - `arrival_notice` (cleaner on the way)
  - `end_of_lease_check` (pre clean checklist)
- Campaign type selector updated to include cleaning options
- Thread linked-record types extended with cleaning-specific options:
  - `checklist`
  - `before_after`
- Vertical defaults config added (`customerconnect.vertical`)

## Notes
- No rewrites. Everything is additive.
- No additional external provider dependencies.
